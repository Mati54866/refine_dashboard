## <a name="introduction">🤖 Introduction</a>

A comprehensive CRM dashboard built with React, featuring secure authentication, interactive charts, sales management, and a fully functional Kanban board with real-time updates. This project demonstrates modern web development practices with a focus on clean UI/UX and responsive design.

## <a name="tech-stack">⚙️ Tech Stack</a>

- **React.js** - Frontend library
- **TypeScript** - Type-safe JavaScript
- **GraphQL** - API query language
- **Ant Design** - UI component library
- **Refine** - React-based framework
- **Vite** - Build tool
- **Codegen** - TypeScript code generation

## <a name="features">🔋 Key Features</a>

- ✅ **Authentication** - Secure login, signup, and password recovery
- ✅ **Authorization** - Granular access control and user permissions
- ✅ **Dashboard** - Interactive charts showing key metrics and real-time updates
- ✅ **Companies Management** - Complete CRUD operations with pagination and search
- ✅ **Kanban Board** - Collaborative task management with drag-and-drop functionality
- ✅ **Account Settings** - Personalized user profile management
- ✅ **Responsive Design** - Fully optimized for all devices

## <a name="quick-start">🚀 Quick Start</a>

### Prerequisites

- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/)
- [npm](https://www.npmjs.com/)

### Installation

```bash
# Clone the repository
git clone https://github.com/Mati54866/refine_dashboard.git

# Navigate to project
cd refine_dashboard

# Install dependencies
npm install

# Start development server
npm run dev
